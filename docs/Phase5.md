# Phase 5: Stock Page

## Goal

Build a TradingView-level stock detail page.

## Features

### Chart

* Candlestick chart
* Line chart toggle
* Time filters (1D, 1W, 1M, 3M, 1Y)
* Live updates via WebSocket

### Stock Info Panel

* Price
* Change %
* High / Low
* Volume
* Market cap

### Tabs

1. Overview
2. News
3. Analysis
4. Financials

### Buy/Sell

* Quantity input
* Instant UI updates
* Toast confirmation

## APIs

* Finnhub (price, chart, news)

## Output

* Real-time interactive stock page
